Potential Questions & Answers

Q: Why use try-except instead of just loading the file?

A: This makes our code more robust. If the file is missing, we get a clear error message instead of the program crashing unexpectedly.

Q: What is the advantage of using aliases (pd, np, plt, sns)?

A: Aliases make the code shorter and more readable. Instead of writing "pandas.read_csv()" we can write "pd.read_csv()".
