Introduction to Data Visualization

Principles, types, and choosing the right chart

Why This Matters

Effective visualization transforms complex data into understandable insights. It helps identify patterns, communicate findings, and drive data-informed decisions. Choosing the right visualization is crucial for conveying your message clearly.

Visualization Principles

  • Clarity Over Creativity

    Prioritize clear communication over decorative elements

  • Know Your Audience

    Tailor complexity to your audience's expertise

  • Highlight Key Insights

    Use visual cues to direct attention to important findings

  • Simplify

    Remove unnecessary elements that don't convey information

Choosing the Right Chart

Purpose Recommended Charts
Compare values Bar chart, Column chart
Show distribution Histogram, Box plot
Reveal relationships Scatter plot, Bubble chart
Display composition Pie chart, Stacked bar
Show trends over time Line chart, Area chart

Matplotlib Fundamentals

Creating basic plots and customizing visuals

Why This Matters

Matplotlib is the foundation of Python visualization. It provides fine-grained control over every aspect of your plots. Mastering Matplotlib enables you to create publication-quality visualizations and forms the basis for understanding higher-level libraries.

Code Example

matplotlib_basics.py 
# Import libraries
import matplotlib.pyplot as plt
import numpy as np

# Nigerian state data
states = ['Lagos', 'Kano', 'Abuja', 'Rivers', 'Oyo']
population = [14.8, 4.3, 3.6, 3.3, 3.2]  # Millions
gdp = [102.4, 23.7, 29.3, 21.6, 16.2]    # Billion USD

# Create figure and axes
fig, ax = plt.subplots(figsize=(10, 6))

# Bar plot for population
x = np.arange(len(states))
ax.bar(x - 0.2, population, 0.4, label='Population (Millions)', color='#006b3f')

# Bar plot for GDP
ax.bar(x + 0.2, gdp, 0.4, label='GDP (Billions USD)', color='#fcd116')

# Customize the plot
ax.set_xlabel('State')
ax.set_ylabel('Value')
ax.set_title('Nigerian State Population and GDP')
ax.set_xticks(x)
ax.set_xticklabels(states)
ax.legend()

# Add grid and improve layout
ax.grid(axis='y', linestyle='--', alpha=0.7)
plt.tight_layout()

# Save and show the plot
plt.savefig('nigerian_states.png', dpi=300)
plt.show()

1. Importing Libraries:
matplotlib.pyplot is for plotting graphs.
numpy helps handle numeric data like x-axis positions.

2. State Data:
We define states, population, and gdp using Python lists.

3. Bar Chart Setup:
fig, ax = plt.subplots() sets up the plotting canvas.
ax.bar() creates the bar charts for both population and GDP.
np.arange gives evenly spaced x-axis points.

4. Chart Customization:
Titles, x/y axis labels, and legend() are added.
grid() makes it easier to read the graph.
plt.tight_layout() avoids label overlap.

5. Save & Show:
plt.savefig() saves the image.
plt.show() displays it on the screen.

Common Mistakes to Avoid

  • Overcrowding the chart with too many elements
  • Using misleading scales or axes
  • Choosing inappropriate chart types
  • Ignoring accessibility (color blindness considerations)
  • Forgetting to label axes or provide context

Advanced Visualization

Seaborn, Plotly, and geographic visualizations

Why This Matters

Advanced visualization libraries help create more sophisticated and interactive visuals. Seaborn provides statistical visualizations, Plotly enables interactivity, and geographic visualizations help understand spatial patterns - crucial for analyzing Nigeria's diverse regions.

Area Chart

Economic growth by region

Heatmap

Population density by state

Geographic Map

Economic activity across Nigeria

advanced_visualization.py 
# Import libraries
import seaborn as sns
import plotly.express as px
import pandas as pd

# Load Nigerian economic data
df = pd.read_csv('nigerian_economy.csv')

# Set Nigerian color palette
nigerian_palette = ['#006b3f', '#fcd116', '#1e40af']
sns.set_palette(nigerian_palette)

# Create a Seaborn pairplot
sns.pairplot(df, hue='Region', diag_kind='kde')
plt.title('Nigerian Economic Indicators by Region')
plt.tight_layout()
plt.show()

# Create an interactive Plotly chart
fig = px.scatter(
    df, 
    x='Population', 
    y='GDP', 
    size='GDP_per_capita', 
    color='Region',
    hover_name='State',
    title='Nigeria: GDP vs Population by State'
)

fig.update_layout(
    xaxis_title='Population (Millions)',
    yaxis_title='GDP (Billions USD)',
    legend_title='Region'
)

fig.show()

# Create a geographic visualization
fig = px.choropleth(
    df,
    geojson='nigeria_states.geojson',
    locations='State',
    featureidkey="properties.name",
    color="GDP",
    color_continuous_scale="Viridis",
    title='GDP Distribution Across Nigerian States'
)

fig.update_geos(fitbounds="locations", visible=False)
fig.update_layout(margin={"r":0,"t":30,"l":0,"b":0})
fig.show()

1. Import Libraries:
seaborn is for advanced plots.
plotly.express is for interactive charts.
pandas helps read and manipulate CSV data.

2. Load CSV:
pd.read_csv() loads Nigerian economic data from a file.

3. Color Styling:
sns.set_palette() sets green/yellow/blue colors based on Nigeriaâ€™s flag.

4. Pairplot:
Shows correlation between multiple indicators using sns.pairplot(), colored by region.

5. Plotly Scatter:
px.scatter() creates an interactive bubble chart.
Each state shows its GDP and population.
size='GDP_per_capita' scales the bubble sizes.

6. Plotly Choropleth:
Visualizes GDP by state on the Nigerian map.
geojson file contains Nigeriaâ€™s state borders.
featureidkey matches state names with shapes.
