Introduction to Pandas

DataFrames, Series, and core concepts

Why This Matters

Pandas is the foundation of data analysis in Python. It provides powerful, flexible data structures (DataFrames and Series) that make data manipulation and analysis efficient. Understanding these core concepts is essential for all data science work.

Code Example

pandas_intro.py 
# Import pandas
import pandas as pd

# Create DataFrame from Nigerian states data
data = {
    'State': ['Lagos', 'Kano', 'Abuja', 'Rivers', 'Oyo'],
    'Population': [14.8, 4.3, 3.6, 3.3, 3.2],  # Millions
    'GDP': [102.4, 23.7, 29.3, 21.6, 16.2],   # Billion USD
    'Region': ['South West', 'North West', 'North Central', 'South South', 'South West']
}

df = pd.DataFrame(data)

# Display the DataFrame
print(df)
print("\nDataFrame shape:", df.shape)
print("Data types:\n", df.dtypes)
import pandas as pd ‚Äì Imports the pandas library for data analysis.
data = {...} ‚Äì Defines a dictionary with keys like 'State', 'Population', etc.
df = pd.DataFrame(data) ‚Äì Converts the dictionary to a DataFrame table.
print(df) ‚Äì Displays the table in console.
print(df.shape) ‚Äì Shows the number of rows and columns.
print(df.dtypes) ‚Äì Displays data types of each column.

Data Cleaning

Handling missing values, duplicates, and inconsistencies

Why This Matters

Real-world data is often messy and incomplete. Data cleaning is the crucial first step in any analysis, accounting for 60-80% of a data scientist's time. Proper cleaning ensures your analysis is accurate and reliable.

Code Example

data_cleaning.py 
# Sample dataset with missing values
data = {
    'City': ['Lagos', 'Abuja', 'Kano', 'Port Harcourt', 'Ibadan'],
    'Temperature': [32.1, None, 35.2, 31.5, None],
    'Humidity': [78, 65, None, 82, 75],
    'Rainfall': [1200, 1100, 900, None, 1150]
}

df = pd.DataFrame(data)

# Data cleaning operations
df_clean = df.copy()
df_clean.drop_duplicates(inplace=True)  # Remove duplicate rows

# Handle missing values
df_clean['Temperature'].fillna(df_clean['Temperature'].mean(), inplace=True)
df_clean['Humidity'].fillna(df_clean['Humidity'].median(), inplace=True)
df_clean['Rainfall'].fillna(df_clean['Rainfall'].mean(), inplace=True)

# Standardize city names
df_clean['City'] = df_clean['City'].str.title()

# Display cleaned DataFrame
print(df_clean)
data = {...} ‚Äì Creates sample data with missing values (None).
df = pd.DataFrame(data) ‚Äì Converts it to a DataFrame.
df_clean = df.copy() ‚Äì Makes a copy to clean without affecting original.
drop_duplicates() ‚Äì Removes duplicate rows if any.
fillna(...) ‚Äì Fills missing values with mean or median of column.
str.title() ‚Äì Formats city names (e.g., "lagos" ‚Üí "Lagos").
print(df_clean) ‚Äì Prints the cleaned data.

Data Cleaning Techniques

  • Handling Missing Values

    Fill with mean, median, mode, or use interpolation

  • Removing Duplicates

    Identify and remove duplicate records

  • Standardizing Values

    Ensure consistent formatting (e.g., capitalization)

  • Data Type Conversion

    Ensure correct data types for each column

Common Data Issues

  • Missing values (NaNs)
  • Inconsistent formatting
  • Outliers and anomalies
  • Incorrect data types
  • Duplicate records

Data Analysis Techniques

Filtering, grouping, aggregation, and descriptive statistics

Why This Matters

Data analysis transforms raw data into meaningful insights. These techniques help you answer critical business questions, identify trends, and make data-driven decisions. Mastering these skills is essential for any data scientist.

data_analysis.py 
# Load Nigerian economic data
df = pd.read_csv('nigerian_economy.csv')

# Filtering data
high_gdp = df[df['GDP'] > 50]  # States with GDP > $50B
south_states = df[df['Region'].str.contains('South')]

# Grouping and aggregation
region_stats = df.groupby('Region').agg({
    'Population': 'sum',
    'GDP': ['sum', 'mean', 'max'],
    'Unemployment': 'mean'
}).reset_index()

# Descriptive statistics
summary = df.describe(include='all')

# Correlation analysis
correlation_matrix = df.corr()
read_csv() ‚Äì Loads data from a CSV file into a DataFrame.
df[df['GDP'] > 50] ‚Äì Filters rows where GDP > $50B.
df[df['Region'].str.contains('South')] ‚Äì Filters states from southern regions.
groupby().agg(...) ‚Äì Groups by region, then aggregates stats like sum and mean.
describe() ‚Äì Summarizes numerical stats of the dataset.
corr() ‚Äì Returns correlation between numerical columns.

Real-World Application

Imagine you're analyzing Nigeria's economic data to advise on resource allocation. Filtering helps identify high-potential states. Grouping by region shows economic disparities. Aggregation reveals total GDP and average unemployment rates. Descriptive statistics provide an overview of economic health. These techniques turn raw data into actionable insights for policymakers.

visualization.py 
# Import visualization libraries
import matplotlib.pyplot as plt
import seaborn as sns

# Set Nigerian color palette
nigerian_colors = ['#006b3f', '#fcd116', '#000000']
sns.set_palette(nigerian_colors)

# Create bar plot of GDP by state
plt.figure(figsize=(10, 6))
sns.barplot(x='State', y='GDP', data=df.sort_values('GDP', ascending=False))
plt.title('GDP by State (Billions USD)')
plt.xticks(rotation=45)
plt.show()

# Create scatter plot of GDP vs Population
plt.figure(figsize=(8, 6))
sns.scatterplot(x='Population', y='GDP', data=df, s=100)
plt.title('GDP vs Population by State')
plt.xlabel('Population (Millions)')
plt.ylabel('GDP (Billions USD)')
plt.show()
import matplotlib.pyplot as plt ‚Äì For charts and graphs.
import seaborn as sns ‚Äì For beautiful statistical plots.
sns.set_palette([...]) ‚Äì Sets color scheme to Nigerian flag colors.
barplot(...) ‚Äì Plots GDP per state as bar chart.
scatterplot(...) ‚Äì Plots GDP vs population as dots on a graph.
plt.show() ‚Äì Displays the chart.
