Array Dimensions

NumPy arrays can have multiple dimensions:

  • 1D: Vector (sequence of values)
  • 2D: Matrix (table of values)
  • 3D: Tensor (cube of values)
  • nD: Higher dimensional arrays

Array Properties

Important array properties:

  • ndim: Number of dimensions
  • shape: Size of each dimension
  • size: Total number of elements
  • dtype: Data type of elements

Performance Benefits

NumPy arrays are more efficient than Python lists:

  • Contiguous memory allocation
  • Vectorized operations
  • Optimized C backend
  • Efficient element access
