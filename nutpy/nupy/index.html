Creating NumPy Arrays

Basic Array Creation

The most straightforward way to create an array is from a Python list or tuple using np.array().

6# Create a 2-dimensional array (matrix)
7arr2d = np.array([[1, 2, 3], [4, 5, 6]])

9# Create a 3-dimensional array
10arr3d = np.array([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])

Key Concepts
  • Dimensions: Arrays can have multiple dimensions (1D, 2D, 3D, etc.)
  • Shape: The structure of the array (e.g., (5,) for 1D, (2,3) for 2D)
  • Elements: All elements in a NumPy array must be of the same type
  • Performance: NumPy arrays are more efficient than Python lists for numerical operations

Special Array Creation

NumPy provides functions to create arrays with specific initial values.

special_arrays.py 
1# Array of zeros
2zeros_arr = np.zeros((3, 4))  # 3 rows, 4 columns
3
4# Array of ones
5ones_arr = np.ones((2, 2, 2))  # 3D array
6
7# Identity matrix
8identity = np.eye(3)  # 3x3 identity matrix
9
10# Array with a range of values
11range_arr = np.arange(0, 10, 2)  # [0, 2, 4, 6, 8]
12
13# Linearly spaced values
14linspace_arr = np.linspace(0, 1, 5)  # [0.0, 0.25, 0.5, 0.75, 1.0]
When to Use Special Arrays
  • zeros(): Initialize arrays before filling with data
  • ones(): Create arrays with initial value 1
  • eye(): Create identity matrices for linear algebra
  • arange(): Generate sequences with fixed steps
  • linspace(): Create arrays with evenly spaced numbers
Common Mistakes
  • Forgetting to import NumPy (import numpy as np)
  • Mismatched dimensions in nested lists
  • Confusing arange() and linspace()
  • Not specifying the shape correctly as a tuple
  • Assuming arrays are mutable like Python lists
