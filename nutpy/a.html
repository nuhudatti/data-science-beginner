2. Special Array Creation Functions

NumPy provides several functions to create arrays with specific initial values.

special_arrays.py 
1# Array of zeros
2zeros_arr = np.zeros((3, 4))  # 3 rows, 4 columns
3
4# Array of ones
5ones_arr = np.ones((2, 2, 2))  # 3D array
6
7# Identity matrix
8identity = np.eye(3)  # 3x3 identity matrix
9
10# Array with a range of values
11range_arr = np.arange(0, 10, 2)  # [0, 2, 4, 6, 8]
12
13# Linearly spaced values
14linspace_arr = np.linspace(0, 1, 5)  # [0.0, 0.25, 0.5, 0.75, 1.0]
15
16# Array with random values
17random_arr = np.random.rand(2, 3)  # 2x3 array with values between 0 and 1
Why These Functions Matter

These array creation functions are essential because:

  • They allow efficient initialization without manual data entry
  • They're optimized for performance and memory usage
  • They create arrays with precise mathematical properties
  • They serve as building blocks for more complex operations

2. Stacking and Splitting Arrays

Combine multiple arrays or split one array into several.

1a = np.array([[1, 2], [3, 4]])
2b = np.array([[5, 6], [7, 8]])
3
4# Vertical stacking
5v_stack = np.vstack((a, b))

7# Horizontal stacking

10# Splitting an array
11arr = np.arange(16).reshape((4, 4))

Important Concept: Views vs Copies

When manipulating arrays, it's crucial to understand when an operation creates a view (shared memory) versus a copy (new memory):

  • Slicing creates views (modifying slice affects original)
  • Reshape usually creates a view if possible
  • copy() method explicitly creates a copy
  • Fancy indexing creates copies
