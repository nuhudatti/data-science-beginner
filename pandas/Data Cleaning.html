2.1 Handling Missing Values

Missing data is a common problem in real-world datasets. Learn strategies to detect, analyze, and handle missing values appropriately.

Critical

Missing Data Visualization

GDP Data 25% missing
Population 12% missing
Oil Production 5% missing
After Cleaning 0% missing
Python
# Load Nigerian economic dataset
import pandas as pd
df = pd.read_csv("nigeria_economy.csv")

# Detect missing values
missing = df.isnull().sum()
print(missing)

# Visualize missing data
import seaborn as sns
sns.heatmap(df.isnull(), cbar=False)

# Handle missing values
# Option 1: Remove rows with missing values
df_clean = df.dropna()

# Option 2: Fill with mean (for numerical data)
df['GDP'] = df['GDP'].fillna(df['GDP'].mean())

# Option 3: Fill with mode (for categorical data)
df['Region'] = df['Region'].fillna(df['Region'].mode()[0])

# Option 4: Forward fill (time-series data)
df['Oil_Production'] = df['Oil_Production'].fillna(method='ffill')

Missing Value Strategies

Detection First

Always analyze missing data patterns before deciding on a strategy. Use isnull().sum() and visualization to understand the extent and pattern of missingness.

Deletion Strategy

Use dropna() when missing values are random and small in number (<5% of dataset). Avoid when it would cause significant data loss.

Imputation Techniques

For numerical data, mean/median imputation works well. For categorical data, use mode. For time-series, forward/backward fill maintains trends.

Advanced Methods

For complex datasets, consider regression imputation, KNN imputation, or multiple imputation techniques.

2.2 Handling Duplicates & Outliers

Identify and remove duplicate records, and detect outliers that might skew your analysis results.

Important
Python
# Identify and remove duplicates
duplicates = df.duplicated().sum()
print(f"Found {duplicates} duplicate rows")

# Remove duplicates based on all columns
df = df.drop_duplicates()

# Remove duplicates based on specific columns
df = df.drop_duplicates(subset=['State', 'Year'])

# Detect outliers using Z-score
from scipy import stats
z_scores = stats.zscore(df['GDP'])
abs_z_scores = np.abs(z_scores)
outliers = (abs_z_scores > 3)
print(f"Found {outliers.sum()} GDP outliers")

# Handling outliers
# Option 1: Remove outliers
df_clean = df[~outliers]

# Option 2: Cap outliers (Winsorization)
q_low = df["GDP"].quantile(0.01)
q_high = df["GDP"].quantile(0.99)
df["GDP"] = np.where(df["GDP"] < q_low, q_low, df["GDP"])
df["GDP"] = np.where(df["GDP"] > q_high, q_high, df["GDP"])

Duplicate & Outlier Management

Duplicate Detection

Duplicates often occur during data collection. Use duplicated() and drop_duplicates() to identify and remove them. Consider which columns should be unique when removing duplicates.

Outlier Detection

Outliers can significantly impact statistical analyses. Z-scores (values beyond ±3 standard deviations) and IQR (Interquartile Range) are common detection methods.

Outlier Treatment

Deciding to remove or transform outliers depends on context. Winsorization caps extreme values at a specified percentile, preserving data while reducing outlier impact.

2.3 Data Type Conversion

Ensure your data is in the correct format for analysis by converting data types and creating new features.

Essential
Python
# Convert data types
df['Year'] = df['Year'].astype('int')        # To integer
df['GDP'] = df['GDP'].astype('float')      # To float
df['Date'] = pd.to_datetime(df['Date'])  # To datetime

# Create categorical types
df['Region'] = df['Region'].astype('category')

# Feature engineering: Extract date parts
df['Year'] = df['Date'].dt.year
df['Month'] = df['Date'].dt.month
df['Quarter'] = df['Date'].dt.quarter

# Create GDP per capita feature
df['GDP_per_capita'] = df['GDP'] * 1e9 / df['Population']  # GDP in billions, population in millions

# Bin continuous variables
bins = [0, 1000, 5000, 10000, 50000, 100000, np.inf]
labels = ['<1B', '1-5B', '5-10B', '10-50B', '50-100B', '>100B']
df['GDP_Category'] = pd.cut(df['GDP'], bins=bins, labels=labels)

Data Transformation Techniques

Data Type Conversion

Correct data types reduce memory usage and enable appropriate operations. Use astype() for basic conversions and pd.to_datetime() for date conversion.

Categorical Data

Convert string columns with limited unique values to categorical type to save memory and enable efficient grouping operations.

Feature Engineering

Create new features from existing data to enhance analysis. Extract date components, calculate ratios (like GDP per capita), and create meaningful categories from continuous variables.
