4.1 Combining Datasets

Learn how to merge, join, and concatenate multiple datasets to create unified data for analysis.

Essential

Concatenation

Stacking DataFrames vertically or horizontally

Merging

Database-style joining based on common columns

Joining

Combining DataFrames based on their indexes

Python
# Load Nigerian economic datasets
import pandas as pd
gdp = pd.read_csv("nigeria_gdp.csv")
population = pd.read_csv("nigeria_population.csv")
oil = pd.read_csv("nigeria_oil_production.csv")

# Concatenation: Stack yearly data vertically
gdp_2010_2015 = pd.read_csv("gdp_2010-2015.csv")
gdp_2016_2020 = pd.read_csv("gdp_2016-2020.csv")
gdp_2021_2023 = pd.read_csv("gdp_2021-2023.csv")
full_gdp = pd.concat([gdp_2010_2015, gdp_2016_2020, gdp_2021_2023])

# Merging: Combine GDP and Population data
economy = pd.merge(gdp, population, on=['State', 'Year'], how='inner')

# Left Join: Add oil production data
economy_full = pd.merge(economy, oil, on=['State', 'Year'], how='left')

# Joining: Combine with region data using index
regions = pd.read_csv("nigeria_regions.csv").set_index('State')
economy_full = economy_full.join(regions, on='State')

# Handle missing values after merge
economy_full['Oil_Production'].fillna(0, inplace=True)

Data Integration Explained

Concatenation

Use pd.concat() to stack DataFrames vertically (axis=0) or horizontally (axis=1). Ideal for combining similar datasets split across files or time periods.

Merging

The pd.merge() function performs database-style joins. Specify how='inner', 'left', 'right', or 'outer' to control which keys to include.

Joining

The join() method combines DataFrames based on their indexes. Useful when you have a primary DataFrame and want to add columns from another.

Post-Merge Handling

After merging, always check for and handle missing values that may result from non-matching keys between datasets.

4.2 Time Series Analysis

Master techniques for working with time-based data, including resampling, rolling windows, and time-based indexing.

COVID-19 impact • GDP: -1.8%

Python
# Convert to datetime and set as index
economy_full['Date'] = pd.to_datetime(economy_full['Year'].astype(str) + '-12-31')
economy_full.set_index('Date', inplace=True)

# Select time periods
q4_2022 = economy_full['2022-10-01':'2022-12-31']
pre_covid = economy_full['2015-01-01':'2019-12-31']

# Resample to quarterly frequency
quarterly_gdp = economy_full['GDP'].resample('Q').mean()

# Rolling averages (12-month moving average)
economy_full['GDP_12MA'] = economy_full['GDP'].rolling(window=12).mean()

# Time-based grouping
annual_growth = economy_full.groupby(pd.Grouper(freq='Y'))['GDP'].sum()
annual_growth_pct = annual_growth.pct_change() * 100

# Shift data for period comparisons
economy_full['GDP_PrevYear'] = economy_full['GDP'].shift(12)
economy_full['GDP_Growth'] = (economy_full['GDP'] / economy_full['GDP_PrevYear'] - 1) * 100

Time Series Concepts

Time-Based Indexing

Set a datetime column as the index to enable powerful time-based selection and slicing using partial datetime strings.

Resampling

Change the frequency of your time series data (e.g., from daily to monthly) using resample(). Specify aggregation methods like mean, sum, or count.

Rolling Windows

Calculate moving averages and other rolling statistics to smooth out short-term fluctuations and highlight longer-term trends.

Period Comparisons

Use shift() to compare values across time periods, such as month-over-month or year-over-year growth rates.
