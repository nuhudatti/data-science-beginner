1.1 Series: One-dimensional Data

A Series is a one-dimensional labeled array capable of holding any data type. Think of it as a single column in a spreadsheet.

Essential
Python
import pandas as pd

# Create a Series from a list
temperatures = pd.Series([28.5, 30.2, 25.8, 32.1], 
                      name="Temperature")

# Create a Series with custom index
cities = pd.Series([28.5, 30.2, 25.8], 
                name="Temperature",
                index=["Lagos", "Abuja", "Kano"])

# Accessing data
print(cities["Lagos"])  # Output: 28.5
print(cities.mean())   # Output: 28.166...

Code Explanation

import pandas as pd

Imports the pandas library and gives it the alias 'pd' for easier reference throughout the code.

temperatures = pd.Series([28.5, 30.2, 25.8, 32.1], name="Temperature")

Creates a Series object - a one-dimensional array with data values. The 'name' parameter labels the entire series.

cities = pd.Series([28.5, 30.2, 25.8], name="Temperature", index=["Lagos", "Abuja", "Kano"])

Creates a Series with custom index labels. This allows accessing elements by label instead of position.

print(cities["Lagos"])

Accesses the value associated with the index label "Lagos" using dictionary-like syntax.

print(cities.mean())

Calculates and prints the average of all values in the Series using the mean() method.

1.2 DataFrames: Two-dimensional Data

A DataFrame is a 2-dimensional labeled data structure with columns that can be of different types, similar to a spreadsheet or SQL table.

Core Concept
Python
# Create DataFrame from dictionary
data = {
    "City": ["Lagos", "Abuja", "Kano", "Port Harcourt"],
    "Population": [21.0, 3.8, 4.3, 1.8],  # in millions
    "GDP": [102.3, 68.4, 28.7, 32.1]   # in billion USD
}

df = pd.DataFrame(data)

# Inspect the DataFrame
print(df.head())      # First 5 rows
print(df.info())      # Structure information
print(df.describe())  # Statistical summary

# Accessing columns
populations = df["Population"]  # OR df.Population

Code Explanation

data = { ... }

Creates a Python dictionary where keys become column names and values become column data.

df = pd.DataFrame(data)

Converts the dictionary into a pandas DataFrame - a 2D tabular data structure with labeled axes.

print(df.head())

Displays the first 5 rows of the DataFrame - essential for initial data inspection.

print(df.info())

Provides metadata about the DataFrame: column count, data types, and memory usage.

print(df.describe())

Generates descriptive statistics for numerical columns: count, mean, std, min, max, etc.

populations = df["Population"]

Selects a single column from the DataFrame, returning it as a Series object.

1.3 DataFrame Operations

Learn how to select, filter, and perform operations on DataFrames to extract valuable insights from your data.

Practical
Python
# Add a new calculated column
df["GDP_per_capita"] = df["GDP"] * 1000 / df["Population"]

# Filter rows based on condition
large_cities = df[df["Population"] > 4]  # Cities with >4M population

# Sorting data
df_sorted = df.sort_values(by="GDP", ascending=False)

# Selecting specific columns
economic_data = df[["City", "GDP", "GDP_per_capita"]]

# Set City as index
df.set_index("City", inplace=True)

Code Explanation

df["GDP_per_capita"] = df["GDP"] * 1000 / df["Population"]

Creates a new column by performing element-wise calculations on existing columns.

large_cities = df[df["Population"] > 4]

Filters rows using a boolean condition, selecting only cities with population >4 million.

df_sorted = df.sort_values(by="GDP", ascending=False)

Sorts the DataFrame based on GDP values in descending order (highest GDP first).

economic_data = df[["City", "GDP", "GDP_per_capita"]]

Selects a subset of columns by passing a list of column names.

df.set_index("City", inplace=True)

Sets the City column as the new index, making city names the primary access method.
