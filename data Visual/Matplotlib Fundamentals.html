Matplotlib Fundamentals

Matplotlib is the foundation of Python data visualization. It provides a MATLAB-like interface and is highly customizable for creating static, animated, and interactive visualizations.

Basic Plot Creation 
import matplotlib.pyplot as plt
import numpy as np

# Create sample data
x = np.linspace(0, 10, 100)
y = np.sin(x)

# Create figure and axis objects
fig, ax = plt.subplots(figsize=(10, 6))

# Plot the data
ax.plot(x, y, 'b-', linewidth=2, label='Sine Wave')

# Customize the plot
ax.set_title('Basic Matplotlib Example', fontsize=14)
ax.set_xlabel('X-axis', fontsize=12)
ax.set_ylabel('Y-axis', fontsize=12)
ax.legend()
ax.grid(True, linestyle='--', alpha=0.7)

# Display the plot
plt.show()

Line-by-Line Explanation:

  • import matplotlib.pyplot as plt

    Imports the Matplotlib plotting library and aliases it as 'plt' for convenience.

  • import numpy as np

    Imports NumPy for numerical operations and array handling.

  • x = np.linspace(0, 10, 100)

    Creates an array of 100 evenly spaced numbers from 0 to 10.

  • y = np.sin(x)

    Calculates the sine of each value in the x array.

  • fig, ax = plt.subplots(figsize=(10, 6))

    Creates a figure and axis object with specified dimensions (10x6 inches).

  • ax.plot(x, y, 'b-', linewidth=2, label='Sine Wave')

    Plots the data with a blue solid line, 2px width, and adds a label for the legend.

  • ax.set_title() / ax.set_xlabel() / ax.set_ylabel()

    Sets the title and axis labels with specified font sizes.

  • ax.legend()

    Displays the legend on the plot.

  • ax.grid(True, linestyle='--', alpha=0.7)

    Adds a dashed grid with 70% opacity.

  • plt.show()

    Displays the final plot.

Key Concepts

  • Figure and Axes: The canvas and plotting areas
  • Plot types: Line, scatter, bar, histogram
  • Customization: Colors, markers, line styles
  • Subplots: Creating multiple plots in one figure

Common Mistakes

  • Forgetting plt.show() in non-interactive environments
  • Overcrowding plots with too much information
  • Not labeling axes or providing context
  • Using default styles without customization

Real Project: Nigerian Health Data

Apply your visualization skills to real Nigerian health data. This project will help you understand health trends across different regions and demonstrate your ability to communicate data insights.

Health Data Visualization 
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Load Nigerian health data
url = "https://data.world/nigeria/health/health-facilities"
health_df = pd.read_csv(url)

# Clean and prepare data
health_df = health_df.dropna(subset=['latitude', 'longitude'])
health_df['facility_type'] = health_df['facility_type'].fillna('Unknown')

# Set up visualization style
sns.set_theme(style="whitegrid", palette="pastel")
plt.figure(figsize=(12, 8))

# Create facility distribution plot
ax = sns.countplot(
    y="state", 
    data=health_df,
    order=health_df['state'].value_counts().index[:10]
)

# Add annotations
for p in ax.patches:
    width = p.get_width()
    ax.text(
        width + 20, 
        p.get_y() + p.get_height()/2,
        '{:1.0f}'.format(width),
        ha='left', 
        va='center'
    )

# Add titles and labels
plt.title('Top 10 Nigerian States by Health Facility Count', fontsize=16)
plt.xlabel('Number of Facilities', fontsize=12)
plt.ylabel('State', fontsize=12)
plt.tight_layout()
plt.show()

Project Implementation Guide:

Data Acquisition:

We're using real Nigerian health facility data from Data World. This dataset contains information about healthcare facilities across all states.

Data Cleaning:

  • Remove records with missing location data
  • Handle missing facility types by marking as 'Unknown'
  • Filter to only include relevant columns for our analysis

Visualization Choices:

Count Plot

Shows distribution of facilities across states

Geographical Plot

Visualizes facility locations across Nigeria

Insight Communication:

This visualization helps policymakers understand healthcare access disparities between states. We can clearly see which states have the most healthcare facilities and which may need additional investment.

Project Requirements

  • Regional Analysis

    Compare healthcare access across Nigeria's six geopolitical zones

  • Facility Types

    Analyze distribution of different facility types (hospitals, clinics, etc.)

  • Interactive Dashboard

    Create an interactive dashboard using Plotly Dash

  • Insight Report

    Summarize key findings and policy recommendations
