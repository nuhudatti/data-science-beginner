Color Theory & Palettes

Color is one of the most powerful tools in data visualization. It can highlight patterns, encode information, and guide the viewer's attention. Understanding color theory is essential for creating effective visualizations.

Color Palette Types

Sequential Palettes

For ordered data from low to high

Diverging Palettes

For data with critical midpoint

Qualitative Palettes

For categorical data with distinct groups

Applying Palettes in Code

Matplotlib Color Customization 
import matplotlib.pyplot as plt
import numpy as np

# Custom color cycle
plt.rcParams['axes.prop_cycle'] = plt.cycler(
    color=['#1f77b4', '#ff7f0e', '#2ca02c', 
           '#d62728', '#9467bd']
)

# Create plot with custom colors
fig, ax = plt.subplots()
x = np.linspace(0, 10, 100)
for i in range(5):
    ax.plot(x, np.sin(x + i * 0.5), 
            linewidth=2)
    
plt.show()

                 '#bc5090', '#ff6361', '#ffa600']

# Create plot with custom palette

sns.boxplot(x="day", y="total_bill", 
            hue="sex", data=tips)

Theme & Style Customization

Professional visualizations require consistent styling. Both Matplotlib and Seaborn provide powerful theming systems to control the overall appearance of your plots.

Using Matplotlib Styles 
# List available styles
print(plt.style.available)

# Apply a style
plt.style.use('ggplot')

# Apply multiple styles (later styles override earlier)
plt.style.use(['dark_background', 'ggplot'])

# Use context manager for temporary style
with plt.style.context('seaborn-whitegrid'):
    fig, ax = plt.subplots()
    ax.plot(x, y)

Seaborn Themes & Context

Seaborn provides high-level control over plot aesthetics through its theming system:

set_theme()

Set aesthetic parameters in one step (style, palette, context)

set_style()

Control plot background and grid styles (darkgrid, whitegrid, etc.)

set_context()

Scale plot elements appropriately for different contexts (paper, talk, poster)

Seaborn Theme Configuration 
# Set overall theme
sns.set_theme(style="whitegrid", 
              palette="pastel",
              context="paper")

# Override specific parameters
sns.set_style("ticks", {"axes.facecolor": "#f0f8ff"})
sns.set_context("talk", font_scale=1.2, 
                rc={"lines.linewidth": 2.5})

# Reset to defaults
sns.reset_orig()

Typography & Text Styling

Text Customization 
# Global font settings
plt.rcParams['font.family'] = 'sans-serif'
plt.rcParams['font.sans-serif'] = ['Segoe UI', 'Tahoma', 'Arial']
plt.rcParams['font.size'] = 12

# Individual text elements
ax.set_title('Nigeria Health Facilities', 
            fontsize=16, 
            fontweight='bold',
            fontfamily='Georgia')

ax.set_xlabel('Year', 
             fontsize=14,
             style='italic')

# Text annotations
ax.annotate('Policy Change', 
           xy=(2018, 4500), 
           xytext=(2016, 5000),
           arrowprops={'arrowstyle': '->', 'color': 'red'},
           bbox={'boxstyle': 'round', 'facecolor': 'wheat', 'alpha': 0.8})

Font Pairing Principles

  • Use sans-serif for labels and numbers
  • Use serif for titles and annotations
  • Maintain consistent font sizes
  • Ensure adequate contrast

Practical Application: Nigerian Health Data

Let's apply these customization techniques to create a publication-quality visualization of Nigerian health facility data.

Customized Health Data Visualization 
import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd
import numpy as np

# Load and prepare data
url = "https://raw.githubusercontent.com/datasets/nigeria-health-facilities/main/data/health-facilities.csv"
df = pd.read_csv(url)
df = df.dropna(subset=['latitude', 'longitude'])
df = df[df['state'].isin(df['state'].value_counts().nlargest(5).index)]

# Set up custom style
plt.style.use('seaborn-whitegrid')
custom_style = {
    'axes.edgecolor': '#4a4a4a',
    'axes.labelcolor': '#2d2d2d',
    'text.color': '#2d2d2d',
    'xtick.color': '#4a4a4a',
    'ytick.color': '#4a4a4a',
    'font.family': ['Segoe UI', 'Tahoma', 'sans-serif'],
    'axes.grid': True,
    'grid.linestyle': '--',
    'grid.alpha': 0.3
}
plt.rcParams.update(custom_style)

# Create figure with constrained layout
fig, ax = plt.subplots(figsize=(12, 8), constrained_layout=True)

# Custom palette
palette = ['#1f77b4', '#ff7f0e', '#2ca02c', '#d62728', '#9467bd']

# Create custom plot
splot = sns.countplot(
    y='state',
    data=df,
    order=df['state'].value_counts().index,
    palette=palette,
    saturation=0.9,
    ax=ax
)

# Add count labels
for p in splot.patches:
    width = p.get_width()
    ax.text(
        width + 20,
        p.get_y() + p.get_height()/2,
        '{:.0f}'.format(width),
        va='center',
        fontsize=10,
        fontweight='bold',
        color='#4a4a4a'
    )

# Customize spines
ax.spines['top'].set_visible(False)
ax.spines['right'].set_visible(False)
ax.spines['left'].set_position(('outward', 10))
ax.spines['bottom'].set_position(('outward', 5))

# Titles and labels
ax.set_title(
    'Health Facility Distribution in Top 5 Nigerian States',
    fontsize=16,
    fontweight='bold',
    pad=20
)
ax.set_xlabel('Number of Facilities', fontsize=12, labelpad=10)
ax.set_ylabel('State', fontsize=12, labelpad=10)

# Add annotation
ax.annotate(
    'Lagos has the highest concentration\nof healthcare facilities',
    xy=(900, 0.1),
    xytext=(600, -1.0),
    arrowprops={'arrowstyle': '->', 'color': '#d62728'},
    fontsize=10,
    bbox={'boxstyle': 'round', 'facecolor': '#f0f0f0', 'alpha': 0.9}
)

# Add footnote
plt.figtext(
    0.5, -0.05,
    'Source: Nigerian Health Facilities Dataset | Visualization: NITDA/NCAIR Cohort 27',
    ha='center',
    fontsize=9,
    color='#6a6a6a'
)

plt.show()
