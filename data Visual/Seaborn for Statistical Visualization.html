Seaborn for Statistical Visualization

Seaborn is a Python data visualization library based on Matplotlib that provides a high-level interface for drawing attractive and informative statistical graphics. It integrates closely with pandas DataFrames.

Why Seaborn?

  • Built-in themes for styling matplotlib graphics
  • Tools for visualizing univariate and bivariate distributions
  • Functions for visualizing linear relationships
  • Statistical estimation and error bars
  • Tools that work well with pandas DataFrames

Seaborn vs Matplotlib

Feature Seaborn Matplotlib
Learning Curve Easier for statistical plots Steeper but more flexible
Default Styles Modern and attractive Basic and functional
Pandas Integration Excellent (DataFrame-native) Requires manual conversion
Statistical Plots Built-in support Requires manual implementation
Use Case Exploratory data analysis Custom publication-quality figures

Core Seaborn Plot Types

Distribution Plots

Visualize the distribution of a dataset. Essential for understanding data spread and central tendencies.

Histogram & KDE 
# Histogram with Kernel Density Estimate
sns.histplot(data=df, x='age', kde=True, bins=20)

# Empirical Cumulative Distribution
sns.ecdfplot(data=df, x='income', hue='region')
Use for: Understanding data distribution, identifying skewness, comparing subgroups

Categorical Plots

Compare different categories and their relationships. Ideal for grouped data analysis.

Boxplot & Violin 
# Box plot showing distribution
sns.boxplot(data=df, x='region', y='income')

# Violin plot showing density
sns.violinplot(data=df, x='education', y='age', 
               inner='quartile')
Use for: Comparing categories, visualizing distributions, identifying outliers

Relational Plots

Visualize relationships between variables. Crucial for correlation and trend analysis.

Scatter & Line Plots 
# Scatter plot with regression line
sns.lmplot(data=df, x='gdp', y='life_expectancy', 
          hue='continent', height=6)

# Line plot with confidence interval
sns.lineplot(data=df, x='year', y='population', 
             hue='country', ci=95)
Use for: Identifying correlations, tracking trends over time, multivariate analysis

Matrix Plots

Visualize matrix-like data. Perfect for correlation matrices and cluster analysis.

Heatmap & Clustermap 
# Correlation matrix heatmap
corr = df.corr()
sns.heatmap(corr, annot=True, cmap='coolwarm')

# Hierarchical clustering heatmap
sns.clustermap(corr, cmap='vlag', standard_scale=1)
Use for: Correlation analysis, pattern recognition, clustering visualization

Practical Application: Nigerian Health Data

Let's apply Seaborn to visualize Nigerian health facility data. We'll explore regional distributions, facility types, and access patterns.

Health Facility Analysis 
import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

# Load Nigerian health facility data
url = "https://raw.githubusercontent.com/datasets/nigeria-health-facilities/main/data/health-facilities.csv"
health_df = pd.read_csv(url)

# Data preparation
health_df = health_df.dropna(subset=['latitude', 'longitude'])
top_states = health_df['state'].value_counts().nlargest(10).index
health_df = health_df[health_df['state'].isin(top_states)]

# Set Seaborn style
sns.set_theme(style="whitegrid", palette="muted")

# Create figure with subplots
fig, axs = plt.subplots(2, 2, figsize=(16, 12))
fig.suptitle('Nigerian Health Facility Analysis', fontsize=18)

# Plot 1: Facility distribution by state
sns.countplot(
    y='state', 
    data=health_df, 
    order=health_df['state'].value_counts().index,
    ax=axs[0, 0]
)
axs[0, 0].set_title('Health Facilities by State')
axs[0, 0].set_xlabel('Number of Facilities')

# Plot 2: Facility types
sns.countplot(
    y='type', 
    data=health_df, 
    order=health_df['type'].value_counts().index,
    ax=axs[0, 1]
)
axs[0, 1].set_title('Facility Type Distribution')
axs[0, 1].set_xlabel('Count')

# Plot 3: Ownership distribution
sns.countplot(
    x='ownership', 
    data=health_df, 
    order=health_df['ownership'].value_counts().index,
    ax=axs[1, 0]
)
axs[1, 0].set_title('Facility Ownership')
axs[1, 0].set_xlabel('')
axs[1, 0].tick_params(axis='x', rotation=45)

# Plot 4: Facility types by state
type_by_state = health_df.groupby(['state', 'type']).size().unstack().fillna(0)
sns.heatmap(
    type_by_state, 
    annot=True, 
    fmt='g', 
    cmap='Blues', 
    ax=axs[1, 1]
)
axs[1, 1].set_title('Facility Types by State')
axs[1, 1].set_xlabel('Facility Type')
axs[1, 1].set_ylabel('State')

# Final adjustments
plt.tight_layout()
plt.subplots_adjust(top=0.92)
plt.show()

Code Explanation:

Data Preparation:

  • health_df = pd.read_csv(url)

    Loads Nigerian health facility data from a public GitHub repository.

  • dropna()

    Removes facilities with missing location data.

  • health_df['state'].isin(top_states)

    Filters to only the top 10 states by facility count.

Visualization Setup:

  • sns.set_theme()

    Sets Seaborn's visual theme to "whitegrid" with a muted color palette.

  • plt.subplots(2, 2)

    Creates a 2x2 grid of subplots for our multi-panel visualization.

Plot Creation:

  • sns.countplot()

    Creates bar charts showing counts of facilities by state, type, and ownership.

  • sns.heatmap()

    Creates a matrix showing facility types distribution across states.

  • .set_title(), .set_xlabel()

    Customizes plot titles and axis labels for clarity.

Insights from Visualization:

Regional Distribution

Reveals which states have the highest concentration of health facilities.

Facility Types

Shows the most common types of health facilities across Nigeria.

Ownership Patterns

Identifies the balance between public and private healthcare providers.

Regional Disparities

Highlights differences in healthcare infrastructure across states.

Seaborn Best Practices

Do's of Seaborn Visualization

  • Use Appropriate Plots

    Match plot types to your data characteristics and analysis goals

  • Leverage Hue Semantic

    Use color (hue) to represent additional variables effectively

  • Contextualize with Titles/Labels

    Always add descriptive titles and axis labels

  • Adjust Figure Size

    Resize plots appropriately for better readability

Don'ts of Seaborn Visualization

  • Overcrowd Plots

    Avoid too many categories or complex visualizations

  • Ignore Colorblind Accessibility

    Use color palettes accessible to colorblind viewers

  • Forget Context

    Don't present visualizations without proper context

  • Misrepresent Data

    Avoid scales or visual encodings that distort data

Choosing the Right Color Palette

Sequential

For ordered data

Diverging

For critical values

Qualitative

For categories
