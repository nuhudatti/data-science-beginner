# 3. DATA CLEANING

# A. Handle missing values
df.fillna(df.median(), inplace=True) # Fill numeric gaps
df.dropna(axis=1, how='all', inplace=True) # Remove empty columns

# B. Handle outliers
for col in df.select_dtypes(include='number'):
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    # Filter values within 1.5*IQR range
    df = df[(df[col] > (Q1 - 1.5 * IQR)) & (df[col] < (Q3 + 1.5 * IQR))]

Cleaning Explained

Missing Values: We fill empty number spots with column averages. For example, if some houses miss "Bedroom Count", we use the typical value (like 3 bedrooms).

Outliers: We remove extreme values. Like houses priced at $100 million when most are $200k-$500k. We keep only values within the "normal range" (1.5× the IQR range).

Why? Dirty data gives false predictions. Clean data = accurate models!
